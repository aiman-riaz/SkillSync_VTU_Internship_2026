Connected Mongoose to MongoDB Atlas and wrote the Report schema. Added a TTL index on expiresAt so old reports auto-delete after 24 hours.


The Background Task: MongoDB runs a background thread every 60 seconds to find and remove expired documents, so it might not disappear the exact millisecond it "expires" [1, 2].
Date Objects: Make sure your expiresAt field is a BSON Date type (or a JavaScript Date object in Mongoose); the index won't work on numbers or strings [1, 2].
Partial Updates: If you ever update the expiresAt value, MongoDB will automatically recalculate the new deletion time [2].
