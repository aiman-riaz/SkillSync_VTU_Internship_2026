To run an Express application on port 5000, you need to configure your application to listen on that specific port using the app.listen() method. This allows you to access your application in a web browser at the URL http://localhost:5000. 
require('express'): Imports the Express module.
const app = express(): Creates an instance of the Express application.
const PORT = ...: Defines the port number, allowing flexibility for different deployment environments.
app.get('/', ...): Defines a route for the home page that sends a "Hello" response.
app.listen(PORT, ...): Starts the server and makes it listen for incoming network requests on the specified port.
