Hands on session about mongodb.
MongoDB is a NoSQL document database.
It stores data in flexible JSON‑like documents.
Instead of tables and rows, MongoDB uses:
• Collections
• Documents
This makes it easy for developers to store complex data.



Two main categories of databases:
1. SQL Databases (Relational)
   Examples: MySQL, PostgreSQL, Oracle
2. NoSQL Databases (Non‑Relational)
   Examples: MongoDB, Cassandra, Redis
MongoDB belongs to the NoSQL database category
