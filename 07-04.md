Functions are reusable blocks of code designed to perform tasks. They improve readability and modularity of programs. Functions can accept parameters and return values. JavaScript supports function declarations and expressions. Arrow functions provide concise syntax alternatives.



Scope determines where variables are accessible within code. Global scope variables are accessible everywhere. Function scope variables exist only inside functions. Block scope applies to let and const variables. Proper scope control prevents accidental overwriting.



Closures occur when functions retain access to outer variables even after execution ends. They preserve lexical environment references internally. Closures enable data privacy and function factories. Many frameworks depend heavily on closures. Understanding closures strengthens advanced logic building.
