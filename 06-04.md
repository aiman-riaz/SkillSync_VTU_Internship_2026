JavaScript variables store data values using keywords var, let, and const. The var keyword has function scope and allows redeclaration. Let has block scope and prevents redeclaration errors. Const defines constant references that cannot be reassigned. Choosing correct variable type improves maintainability.



JavaScript supports primitive data types including string, number, boolean, undefined, null, bigint, and symbol. Non-primitive types include objects, arrays, and functions. Primitive values store directly in memory. Objects store references instead. Understanding this difference avoids unexpected bugs.



Operators perform operations on variables and values. Arithmetic operators include addition, subtraction, multiplication, and division. Comparison operators evaluate logical relationships between values. Logical operators combine boolean expressions efficiently. Assignment operators simplify variable updates.
