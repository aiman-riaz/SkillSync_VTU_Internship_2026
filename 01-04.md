Built the Home page with a hero section, search bar and example username chips for quick testing. Added decorative background blobs, a grid overlay and a gradient title. Feature cards at the bottom explain what the app does before a user commits to searching.



The blob effect is created using CSS radial gradients with a heavy blur filter applied on top. Two absolutely positioned div elements sit in a fixed background layer behind all the content using z-index. The blur value is set high enough that the shapes lose all hard edges and bleed into each other, creating a soft ambient glow rather than distinct shapes. This technique is common in modern UI design because it adds depth and visual interest without adding any image assets or JavaScript.



The example username chips serve a UX purpose beyond just looking good — they lower the barrier to entry for a first-time visitor who might not know which GitHub username to try. By clicking torvalds or gaearon the user immediately sees a fully populated report with real data, which demonstrates the app's value much faster than an empty search box would. This is a pattern used in search-heavy products to onboard users without requiring them to know what to search for first.
