React Fiber

* React Fiber is the new core algorithm of React.
* It improves rendering speed and responsiveness.
* Breaks work into smaller units for better scheduling.
* It is an internal architecture improvement.

---

Keys in React

* Keys uniquely identify list elements.
* Help React track changes efficiently.
* Improve rendering performance.
* Keys should be unique among siblings.

---

Using Index as Keys

* Using index as keys is not recommended when list order changes.
* Can cause performance issues and incorrect UI behavior.
* React uses index as default if no key is provided.

---

Exports in JavaScript

* A file can have only one default export.
* A file can have multiple named exports.
* Both can exist in the same file.

---

React Hooks

* Hooks are functions provided by React with built-in logic.
* They allow state and lifecycle features in functional components.
* Hooks simplify component logic.
