Requirement analysis is the process of identifying what the system needs to do before writing code. Database schema design involves deciding how data is organized into collections and how documents reference each other. Mongoose is an ODM (Object Data Modeling) library that lets you define schemas for MongoDB collections and work with them as JavaScript objects. dotenv is used to keep sensitive config like database URIs out of the source code. Axios is an HTTP client library for making API calls from React. Setting up the Axios instance early means you only configure base URL and headers once.
