Scaffolded the React + Vite frontend with React Router. Set up index.html with Google Fonts and OG meta tags, wrote global CSS with color variables and animations.


Build Tooling (The Vite Advantage)
The theory behind Vite is Native ESM (ES Modules).
The Problem: Older tools (like Create React App) bundled your entire app before showing you anything, which got slower as the project grew.
The Solution: Vite only serves the code you are currently looking at. It uses "Hot Module Replacement" (HMR) to update the browser instantly, making the feedback loop between writing code and seeing a change nearly zero.


Client-Side Routing
Using React Router introduces the Single Page Application (SPA) theory.
The Goal: Instead of the browser asking the server for a new HTML page every time you click a link, React Router intercepts the click and just swaps out a piece of the page.
The Result: This makes the app feel like a desktop "app" rather than a "website," because transitions are instant and the state (like a user's search query) doesn't reset.
