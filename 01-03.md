JSX (JavaScript XML)

* JSX allows writing HTML-like syntax inside JavaScript.
* Browsers do not understand JSX directly.
* JSX gets converted into React.createElement.
* It improves readability and developer experience.

---

Babel

* Babel is a JavaScript compiler.
* It converts modern JavaScript into browser-compatible code.
* It also converts JSX into JavaScript objects.
* Conversion flow: JSX → React.createElement → React Element → HTML render.

---

Functional Components

* Functional components are JavaScript functions returning JSX.
* Component names must start with uppercase letters.
* Lowercase names are treated as HTML elements.
* Components can be reused easily.

---

Component Composition

* Using one component inside another component is called composition.
* Helps create complex UI from small components.
* Encourages modular design.
